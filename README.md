# 婚纱高级定制试衣旅程页

面向新娘、顾问和版师三个角色的多端协同婚纱定制体验系统。通过可视化的试衣旅程，让用户理解每一次修改的原因、对拍摄的影响以及交付节点，提升高级定制的透明感和仪式感。

## 原始需求

> 请制作婚纱高级定制试衣旅程页，Vue3 页面用于新娘手机、门店试衣平板和顾问大屏，围绕婚期、仪式场地、身形数据、面料样卡、裙型轮廓、头纱配饰、预算、修改轮次和拍摄计划展开。新娘可以在页面中查看 A 字、鱼尾、缎面、蕾丝、拖尾和可拆袖的组合效果，顾问记录试穿感受、行动难度、仪式动线和家人意见，版师看到肩宽、腰臀、鞋跟高度、孕期变化和交付节点。页面要突出视觉质感、试衣动线、前后对比、照片批注和移动端阅读，不能做成普通预约表；用户应能理解每一次修改为什么发生、会影响哪次拍摄、是否赶得上婚礼。

## 启动方式

### 前置要求

- Node.js >= 18.0.0
- npm 或 pnpm
- Docker & Docker Compose（如需容器化部署）

### 本地开发启动

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务

```bash
npm run dev
```

访问地址：http://localhost:5173

#### 3. 类型检查

```bash
npm run check
```

#### 4. 构建生产版本

```bash
npm run build
```

#### 5. 预览生产版本

```bash
npm run preview
```

### Docker 一键启动

#### 1. 构建并启动

```bash
docker compose up --build
```

如需后台运行：

```bash
docker compose up --build -d
```

#### 2. 访问地址

启动后访问：http://localhost:8080

#### 3. 停止服务

```bash
docker compose down
```

## 项目简介

### 核心功能

1. **试衣旅程时间线** - 婚期倒计时、仪式场地、修改轮次进度、拍摄计划节点
2. **身形数据面板** - 肩宽、腰臀比例、鞋跟高度、孕期变化、关键尺寸记录
3. **款式组合预览** - A字/鱼尾裙型、缎面/蕾丝面料、拖尾长度、可拆袖效果
4. **面料样卡展示** - 材质纹理、光泽度、垂坠感、色卡选择
5. **配饰搭配** - 头纱、手套、首饰、婚鞋组合效果
6. **试穿记录** - 顾问记录试穿感受、行动难度、仪式动线评估
7. **照片批注** - 前后对比、局部标注、修改意见可视化
8. **预算管理** - 基础价、修改费用、配饰费用、总预算进度

### 三端视图

- **新娘端（手机）** - 查看试衣进度、浏览款式组合、确认修改方案、查看拍摄计划
- **顾问端（平板）** - 记录试穿感受、标注照片、管理修改轮次、同步家人意见
- **版师端（大屏）** - 查看身形数据、修改节点、交付时间线、工艺细节

### 技术栈

- **前端框架**：Vue 3 + TypeScript + Composition API
- **构建工具**：Vite
- **路由管理**：Vue Router 4
- **状态管理**：Pinia
- **样式方案**：Tailwind CSS 3
- **图标库**：Lucide Vue Next
- **数据方案**：前端 Mock 数据

### 页面结构

| 路由 | 页面 | 说明 |
|------|------|------|
| / | 试衣旅程首页 | 时间线、概览、款式预览 |
| /dress-style | 款式组合 | 裙型、面料、拖尾、袖型组合预览 |
| /body-data | 身形数据 | 尺寸记录、变化曲线、孕期标注 |
| /fitting-records | 试穿记录 | 照片墙、前后对比、顾问笔记 |
| /accessories | 配饰搭配 | 头纱、首饰、婚鞋搭配 |
| /budget | 预算管理 | 费用明细、支付进度 |
| /shooting-plan | 拍摄计划 | 拍摄节点与修改轮次关联 |

## 设计特色

- **优雅高级视觉**：玫瑰金 + 象牙白 + 香槟金配色，丝绸质感背景
- **三端自适应**：新娘手机端、顾问平板端、版师大屏端一键切换
- **沉浸式体验**：优雅的过渡动画、柔和的光影效果
- **信息透明**：每一次修改都清晰展示原因、影响拍摄、交付影响
- **移动端优先**：针对手机竖屏优化，触控友好
